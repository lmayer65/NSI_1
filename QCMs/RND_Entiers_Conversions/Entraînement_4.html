Q.9

Q.10

  • B — 0001 1010
  • C — 1110 0110
  • D — 1110 0111

Corrigé réponses correctes

Q1 → C  |  Q2 → B  |  Q3 → A  |  Q4 → C  |  Q5 → D
Q6 → D  |  Q7 → B  |  Q8 → C  |  Q9 → C  |  Q10 → D

Vérif. rapide : 64 → 0100 0000 ; 0010 1100 → 2C ; 106 → 6A ; 3F₁₆ = 63 ; 2A₁₆ (42) + 2 = 44 ;
−42 : 0010 1010 → inversion 1101 0101 + 1 → 1101 0110 ; −12 → 1111 0100 ; −108 → 1001 0100 ; −25 → 1110 0111.
